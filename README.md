#Endless

##Endless browser tabs turn into an Endless reading list

```
Endless stores the endless list of browser tabs you keep open for later reading

Usage:
  endless [flags]
  endless [command]

Available Commands:
  add         Add an URL to the Endless list
  get         Get an URL from the Endless list
  help        Help about any command
  list        List all URLs from the Endless list
  random      Get a random URL

Flags:
  -h, --help   help for endless

Use "endless [command] --help" for more information about a command.
```
